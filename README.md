Mastodon Notifier
=================

Provides Mastodon integration for Symfony Notifier.

DSN example
-----------

```
MASTODON_DSN=mastodon://ACCESS_TOKEN@HOST
```

where:
 - `ACCESS_TOKEN` is your Mastodon access token
 - `HOST` is your Mastodon host
